# Jaso-fixer
맥OS에서 작성한 파일의 제목이 윈도우에서 깨져 보이는 현상 수정하는 프로그램

----------------------- Update Note -----------------------




### v1.0.1
하위까지 전부 지원.


### v1.0.0
자소분리가 되어있는 파일들을 Drag & Drop하면 파일명 수정.\n
아직 폴더는 미지원. 파일 Ctrl+A를 이용해서 대량의 파일 한번에 진행하기 바람.
